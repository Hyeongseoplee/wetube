# Youtbe clone

<!-- Global router -->

/ -> Home
/join -> Join
/login -> Login
/search -> Search

<!-- user router -->

/user/:id -> See my profile
/user/logout -> Log Out
/users/edit -> Edit my profile
/users/delete -> Delete user

<!-- video router -->

/videos/:id -> See videos
/videos/:id/edit -> Edit video
/videos/:id/delete -> Delete video
/videos/upload -> Upload video
